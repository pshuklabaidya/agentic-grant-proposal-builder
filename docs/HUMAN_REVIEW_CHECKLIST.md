# Human Review Checklist

## Before Submission

Use this checklist before treating any generated proposal as submission-ready.

## Source Completeness

- Full RFP or NOFO uploaded
- Funder FAQ uploaded if available
- Scoring rubric uploaded if available
- Budget instructions uploaded
- Eligibility rules reviewed
- Deadline reviewed
- Required attachments reviewed

## Proposal Narrative

- Executive summary matches funder priorities
- Need statement includes current evidence
- Target population is specific
- Project design is feasible
- Implementation timeline is realistic
- Outcomes are measurable
- Evaluation plan has data sources
- Sustainability plan is credible
- Organizational capacity is supported

## Budget

- Line items sum to requested amount
- Costs are allowable
- Indirect costs follow funder rules
- Personnel assumptions are realistic
- Participant support is justified
- Evaluation costs are appropriate
- Budget narrative matches proposal activities

## Compliance

- Eligibility confirmed
- Required sections included
- Page limits checked
- Formatting rules checked
- Required attachments listed
- Signatures and approvals identified
- Submission portal requirements checked

## AI Output Review

- Unsupported claims removed
- Overconfident language softened
- Evidence snippets inspected
- Reviewer findings resolved
- Quality-gate issues addressed
- Final text reviewed by a human
